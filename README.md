# Project Title

Never Alone is a Gamifictaion of parallel play with a timer option.

## Description

It is a Multiuser Plattform that lets you create rooms to connect with your friends, to get your acitivities done at the same time, without the distraction of a more complex medium.

## Getting Started

Its based on the express Server, Backend is node.js uses websockets and redis as a temporary database, its an university project, so far from perfect

### Dependencies

* A Linux Docker Installation, native on Linux or in WSL

### Installing

* git clone the project on your server

### Executing program

* Start the docker containers with
```
docker compose up (-d)
```
